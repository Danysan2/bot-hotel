🚀 INICIO RÁPIDO - VIAJES COLOMBIA TOURS CHATBOT
================================================

📋 PASOS PARA COMENZAR:

1. INSTALAR DEPENDENCIAS
   pip install -r requirements.txt

2. CONFIGURAR GOOGLE SHEETS
   - Crear hoja de cálculo en Google Sheets
   - Compartir con cuenta de servicio
   - Copiar ID de la hoja
   - Guardar credenciales en service_account.json

3. CONFIGURAR VARIABLES DE ENTORNO
   Crear archivo .env con:
   
   GOOGLE_SHEET_ID=tu_id_aqui
   EVOLUTION_API_URL=tu_url_aqui
   EVOLUTION_API_KEY=tu_key_aqui
   INSTANCE_NAME=tu_instancia
   PORT=8000

4. INICIAR EL SERVIDOR
   python start.py

5. CONFIGURAR WEBHOOK
   python configurar_webhook.py

6. PROBAR EL CHATBOT
   python test_chatbot_viajes.py
